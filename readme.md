Висновки (Monte Carlo для кидання двох кубиків)

У даній роботі було реалізовано метод Монте-Карло для моделювання процесу кидання двох гральних кубиків. Було проведено 1 000 000 симуляцій, у кожній з яких обчислювалась сума значень на двох кубиках.

На основі результатів симуляції були обчислені ймовірності появи кожної можливої суми від 2 до 12. Отримані експериментальні ймовірності було порівняно з аналітичними значеннями, відомими з теорії ймовірностей.

Спостереження:

Симульовані ймовірності дуже близькі до аналітичних

Найбільша ймовірність спостерігається для суми 7, що відповідає теоретичним розрахункам

Відхилення між симульованими та аналітичними значеннями є незначними та зменшуються при великій кількості симуляцій

Висновок:

Метод Монте-Карло ефективно апроксимує ймовірнісні характеристики випадкових процесів. При достатньо великій кількості ітерацій результати симуляції збігаються з аналітичними значеннями, що підтверджує коректність реалізації алгоритму та практичну цінність методу.
